Color Segmentation

This was an idea developed by me and my coworker Luan Pontolio to help people who work with image processing. With this project, it is possible to select one or more color interval and apply the segmentation of areas containing these intervals to one or more images. The purpose is to segment images by color intervals.

Materialize Colorpicker

It refers to a color picker using Materialize CSS. The original source was called bootstrap-colorpicker and it used bootstrap to stylize the component. In my version, Materialize is used instead of bootstrap to give this component the material design.

SageOne

Spotify Gem

Gem to access Spotify API. While I was developing the Monitoring Emotions system (listed on private section of my portfolio), I had some difficulties to make contact with Spotify and play songs.

For this reason I decided to work on a Ruby solution to supply features that another gems didn't provide me, like timeout. This gem is not finished yet, Pull Requests are welcome.
